Cloudflare & Public Access

DNS (Cloudflare)

loganlancer.com     CNAME   <tunnel-uuid>.cfargotunnel.com   Proxied
www.loganlancer.com  CNAME   <tunnel-uuid>.cfargotunnel.com   Proxied
# Nameservers: Cloudflare
# SSL/TLS: Full (edge terminates TLS)
# Always Use HTTPS: On

Tunnel config (origin HTTP)

tunnel: webserver
credentials-file: /root/.cloudflared/webgw.json
ingress:
  - hostname: loganlancer.com
    service: http://localhost:80
  - hostname: www.loganlancer.com
    service: http://localhost:80
  - service: http_status:404

Apache vhost (origin)

<VirtualHost *:80>
  ServerName loganlancer.com
  ServerAlias www.loganlancer.com
  DocumentRoot /var/www/html
  # No HTTPS redirects; Cloudflare handles TLS and canonicals.
</VirtualHost>
