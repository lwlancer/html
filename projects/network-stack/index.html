Overview

This network stack powers all my self-hosted projects—from virtual machines to AI models and media servers. It’s built around a Proxmox VE cluster with multiple lightweight Linux containers running specialized services.

Core Services

  • Proxmox Cluster: Central virtualization environment managing all containers and VMs.
  • Apache Webserver: Hosts this site and project documentation.
  • Cloudflare Tunnel: Provides secure, resilient external access without open inbound ports.
  • Home Monitoring: Grafana and Prometheus track resource usage across the cluster.
  • Media Stack: Jellyfin streams personal media throughout the network.
  • LLM Stack: Local Ollama instance and Piper/Whisper containers for voice interaction.
  • Game Server: Self-hosted Minecraft world for testing performance and networking.
  • VPN and Proxy Layer: OpenVPN and SOCKS5 for secure internal routing and external testing.

Recent Upgrades

  • Replaced direct HTTPS with a Cloudflare Tunnel for reliable access through NAT and CGNAT.
  • Moved DNS management to Cloudflare for automatic certificate provisioning and routing.
  • Rebuilt the webserver using Apache inside a clean Ubuntu LXC.
  • Stabilized Proxmox bridge networking and LXC configurations.

Future Plans

  • Container orchestration for auto-recovery and scaling.
  • Centralized logging with Loki and Promtail.
  • Custom dashboard displaying live cluster health directly on this site.
